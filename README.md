# PLD Agile

**Hexanôme H21**

## Utilisation

Compiler le code principal :
```bash
javac -d target/classes src/main/java/model/*.java
```

Compiler les tests :
```bash
javac -cp target/classes -d target/test-classes src/test/java/model/*.java
```

Exécuter les tests :
```bash
java -cp target/classes model.<classTest>
```
